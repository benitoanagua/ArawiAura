Sometimes the code breaks:  
`undefined is not a feeling`,  
`cannot read property 'tomorrow' of undefined`,  
`syntax error in heart.js`.

I refactor my soul,  
comment out the pain,  
push to production  
a version that compiles,  
that runs without errors,  
that returns `true` for happiness.
